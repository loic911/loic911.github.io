RestApiDoc plugin

The RestApiDoc plugin allow you to document your Grails Rest API. Thanks to some Annotations (@), you will be ready to build a full API report (with a playground to perform test request). The plugin is based on json-doc (linkkk).

See source code »

RestApiDoc plugin. A little presentation.

This plugin allow you to document actions/methods (description, HTTP path/verb, parameters and their type, response type...) and resources (description, all fields,...).

A sample report is available here. The github repo for this sample is here.
